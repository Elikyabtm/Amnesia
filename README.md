# Amnesia

**Amnesia** est un jeu d’enquête numérique interactif dans lequel le joueur doit retrouver son identité en explorant des traces laissées dans un environnement numérique.

Le jeu plonge le joueur dans une interface simulant un système informatique. En fouillant dans des e-mails, des fichiers, des photos et différents documents, il doit reconstituer progressivement l’histoire du personnage et comprendre qui il est réellement.

L’expérience repose sur l’exploration, l’observation et la déduction.

---

## Concept

Le joueur se réveille face à un ordinateur sans aucun souvenir de son identité.

L’unique piste disponible se trouve dans les données numériques accessibles depuis l’ordinateur. Chaque élément consulté — message, document, image ou information personnelle — constitue un indice permettant de reconstruire une histoire et de révéler progressivement l’identité du personnage.

Certaines informations peuvent orienter vers différentes interprétations, invitant le joueur à analyser attentivement les traces numériques pour comprendre la vérité.

---

## Gameplay

Le jeu repose sur l’exploration d’un environnement numérique interactif.

Le joueur peut notamment :

- consulter des e-mails  
- explorer des dossiers et des documents  
- analyser des images et des fichiers personnels  
- découvrir des indices narratifs cachés dans l’interface  

Ces éléments permettent de reconstituer progressivement l’identité du personnage et de comprendre les événements liés à son passé.

---

## Technologies utilisées

Le projet est développé avec les technologies suivantes :

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**

---

## Architecture du projet

    app/        Pages principales et configuration globale de l’application
    components/ Composants React utilisés pour construire l’interface du jeu
    public/     Ressources statiques (images, icônes, éléments visuels)
    styles/     Styles globaux et configuration visuelle

---

## Objectif du projet

Amnesia explore la manière dont les traces numériques peuvent raconter l’histoire d’une personne.

À travers une interface interactive, le joueur assemble progressivement des fragments d’informations pour reconstituer une identité et comprendre ce qui s’est réellement passé.
